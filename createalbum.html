Alright! Let's make our first album. Let's start by clicking the upload button and selecting an image or two. You'll see the images appear on your left.

To create group tags, type your tag name here, these tags will be added to every image in this album, including ones you add later. You can upload and tag images in whatever order you feel like.

To add a tag to one particular image only add it in here

If we want to remove a tag from one particular image click the "x" over here.

If we want to delete a tag from all the groups click the "x" over here

When You're done, Hit save and then done or just click done, to exit with out saving.
